# 2. 项目概述

## 2.1 产品描述

通过开发基于微信平台的辅助查寝小程序帮助辅导员减轻晚间查寝的工作量，及时了解学生晚上是否在宿舍的情况，以确保学生的安全。

## 2.2 产品功能

辅导员工作比较繁忙，对于晚间查寝往往会心有余而力不足，而现有的查寝方式因为种种原因，辅导员无法及时得到反馈。

辅助查寝系统致力于通过微信平台辅助辅导员进行查寝工作，大大减轻辅导员工作量。首先通过用例图了解角色分工如下：
